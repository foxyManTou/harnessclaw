### 新增

- 对话页产物文件路径现在以内联文件 chip 形式展示，可直接点击查看交付物。
- HTML 产物文件支持通过 iframe 实时预览为可交互页面。
- 通用模式新增产物展示，配套优化侧边栏与输入框 UI。
- 侧边栏恢复 x-Lab 导航入口。
- 对话回合运行期间，工作区文件数量自动刷新。
- 新增会话侧边面板与首页问号图标，优化会话侧边栏与首页布局。
- Agent Team 标签页新增预览提示横幅，并明确自定义 Agent 的适用范围。

### 变更

- 对话页顶部三点菜单恢复「重命名会话」功能。
- 优化对话页标题与对话框样式。
- 工具结果网格中的生成图片现在保留原始宽高比。
- 首页「24h Online」状态徽标替换为设计图标。
- 引导流程的任务画像改为可选且更易发现，并在头部提供显式的 中文 / English 切换。
- 发布工作流对所有版本（含 beta）统一使用最新引擎运行时，移除严格版本号匹配。

### 修复

- 埋点上报改用与服务端兼容的 HMAC 签名请求头，修复 401/404，恢复生产埋点投递。
- 修复助手回复中生成图片链接渲染：本地 `file://` 与绝对路径产物现通过安全本地文件协议展示，不再显示损坏图片。
- Browser Agent 会话运行在独立 helper 进程并隔离 CDP target，避免误接管主窗口或其他浏览器会话。
- 移除对话页输入框聚焦时的蓝色边框，对齐主页样式。
- 修复日志栏秘书头像/名称/LOGO 切换页面后错乱的问题。
- 加固已安装技能（skills）的加载器并明确调用方式。
- 在已禁用的提供方上切换模型时跳过引擎同步。
- 修复安全依赖告警（dependabot）。
